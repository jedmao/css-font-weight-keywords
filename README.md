# css-font-weights

List of CSS [font weights](https://developer.mozilla.org/en-US/docs/Web/CSS/font-weight):
- normal
- bold
- lighter
- bolder
- 100, 200, 300, 400, 500, 600, 700, 800, 900

## Installation

```
npm install css-font-weights [--save[-dev]]
```

## Usage

```js
var cssFontWeights = require('css-font-weights');
cssFontWeights; /*
[
	"normal",
	"bold",
	"bolder",
	"lighter",
	"100",
	"200",
	"300",
	"400",
	"500",
	"600",
	"700",
	"800",
	"900"
]
*/
```
